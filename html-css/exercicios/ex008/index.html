Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase temos um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase temos um termo em ênfase usando a tag <em> (semântica).

Texto marcado

Nesta frase temos um termo marcado usando a tag <mark>.

Texto GRANDE e pequeno

Nesta frase temos um termo grande usando a tag <big> e um termo pequeno usando a tag <small>.

Texto deletado

Nesta frase temos um termo deletado usando a tag <del>.

Texto sublinhado / inserido

Nesta frase temos um termo sublinhado usando a tag <u> (não semântica).

Nesta frase temos um termo inserido usando a tag <ins> (semântica).

Texto sobrescrito

Nesta frase temos um termo sobrescrito usando a tag <sup>.

Texto subscrito

Nesta frase temos um termo subscrito usando a tag <sub>.
